# How to Start

## 초기 환경설정
### 주의: node 12버전이상
### yarn package 설치
- yarn 설치 후 진행
- package.json 패키지 설치 
```shell
yarn install
```
## 시작하는 법

### 1. 로컬 환경

```shell
yarn start
```
